How to Run the Build Script

1. Prerequisites:
   - Ensure Python is installed on your system.
   - Ensure you have the "unittest" module available.

2. Steps to Run:
   - To clean previous builds: run `make clean`.
   - To run all unit tests: run `make test`.
   - To build the application: run `make build`.

3. Output:
   - The built files will be located in the 'dist' folder.
